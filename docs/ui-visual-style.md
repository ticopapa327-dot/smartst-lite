## 1. 结论

视捷UST 新阶段界面必须采用 `or-preview HMI palette v0.3`：冷灰蓝医疗设备控制屏风格。

该视觉标准适用于：

- 手术室 Windows 工作台。
- 示教室 Windows 客户端。
- Android 会议平板客户端。
- 手机 H5 单向观察页面。
- 录像、文件、设置等需要保持产品一致性的页面。

后台或配置页面可以降低控件尺寸，但不得改成互联网后台、BI 驾驶舱、大面积蓝色科技屏或紫蓝渐变风格。

## 2. 设计基准

- 基准分辨率：`1920x1080`。
- 基准场景：浏览器或桌面应用全屏触控终端。
- 参考方向：高端医疗设备控制屏的冷灰、低饱和蓝、实体按键视觉。
- 法务边界：只参考色系和控制终端气质，不复制第三方医疗设备的界面结构、图标、文案和素材。

## 3. CSS Token

| 用途 | CSS 变量 | 色值 | 使用规则 |
| --- | --- | --- | --- |
| 主背景 | `--hmi-bg` | `#1b1e23` | 页面主体背景，整体必须偏暗 |
| 左侧导航/深底 | `--hmi-rail` | `#121417` | 导航栏、深色底座，不做亮色导航 |
| 常规面板 | `--hmi-panel` | `#252a30` | 控制卡片、信息区、弹层主体 |
| 强面板/停止键 | `--hmi-panel-strong` | `#313840` | 停止键、重点暗色面板、局部强调 |
| 默认控件 | `--hmi-control` | `#30363d` | 信号源卡片、输出卡片、实体按键默认态 |
| 控件 hover | `--hmi-control-hover` | `#3a414a` | 鼠标/触控反馈，不加发光 |
| 当前激活 | `--hmi-control-active` | `#35bce7` | 当前信号源、当前输出目标、焦点状态 |
| 当前导航 | `--hmi-nav-active` | `#6678d8` | 左侧当前功能，仅用于导航选中 |
| 禁用控件 | `--hmi-control-disabled` | `#24282d` | 不可用按钮、无权限状态 |
| 常规边框 | `--hmi-line` | `#41474f` | 面板与控件边界 |
| 强边框 | `--hmi-line-strong` | `#5b626b` | 预览舞台、重点容器边界 |
| 主文字 | `--hmi-text` | `#f3f6f9` | 标题、按钮主文本 |
| 次级文字 | `--hmi-muted` | `#b3bbc3` | 辅助说明、分辨率、状态说明 |
| 医疗蓝强调 | `--hmi-accent` | `#35bce7` | 图标、当前态、小面积强调 |
| 正常状态 | `--hmi-green` | `#58c77c` | 小圆点、在线状态，不做大面积背景 |
| 信息状态 | `--hmi-blue` | `#6faee0` | 辅助状态，不替代医疗蓝主强调 |
| 警告状态 | `--hmi-amber` | `#d9a94a` | 低频警告、小面积标识 |
| 故障状态 | `--hmi-red` | `#df6a63` | 失败、异常、小面积标识 |

## 4. 使用规则

- 黑灰色占主导，医疗蓝只用于选中、激活、焦点和少量图标。
- 面板与背景只形成轻微层级，不做强烈互联网卡片风格。
- 按钮必须接近医疗设备实体按键：深灰底、清晰边框、足够触控尺寸、状态明确。
- 状态色只用于小面积状态提示，不用于整块卡片背景。
- 新增页面优先复用以上 token；确需新增颜色时，必须补充到本文档并说明用途。
- Android 会议平板和手机 H5 必须复用同一色板，只调整布局密度和触控尺寸。

## 5. 禁止项

- 禁止大面积蓝色背景。
- 禁止蓝色导航栏。
- 禁止紫蓝渐变。
- 禁止霓虹发光。
- 禁止玻璃拟态。
- 禁止 BI 驾驶舱式大屏视觉。
- 禁止用状态色铺满整块卡片。
- 禁止复制第三方医疗设备界面结构、图标、文案和素材。

## 6. 前端落地要求

第一轮前端开发必须先在全局 CSS 中建立 HMI token：

```css
:root {
  --hmi-bg: #1b1e23;
  --hmi-rail: #121417;
  --hmi-panel: #252a30;
  --hmi-panel-strong: #313840;
  --hmi-control: #30363d;
  --hmi-control-hover: #3a414a;
  --hmi-control-active: #35bce7;
  --hmi-nav-active: #6678d8;
  --hmi-control-disabled: #24282d;
  --hmi-line: #41474f;
  --hmi-line-strong: #5b626b;
  --hmi-text: #f3f6f9;
  --hmi-muted: #b3bbc3;
  --hmi-accent: #35bce7;
  --hmi-green: #58c77c;
  --hmi-blue: #6faee0;
  --hmi-amber: #d9a94a;
  --hmi-red: #df6a63;
}
```

验收时需要检查：

- 页面截图整体是否以冷灰/深灰为主。
- 医疗蓝是否只用于当前态和小面积强调。
- 按钮是否有足够触控面积和清晰边框。
- 是否出现禁用项中的视觉风格。
- 手机 H5 页面是否因为临时性而偏离产品色板。
